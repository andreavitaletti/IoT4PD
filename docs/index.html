Introduction

This is probably the most simple flow. An inject block (light blue) injects a msg with the timestamp in its payload (i.e. in msg.payload) when pushed. When the msg reaches the debug block (green) the content of the msg.payload (i.e. the timestamp) is shown in the debug window on the right.

The inject block can inject a number of messages
